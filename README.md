# UFC Fight Prediction Project

Проект по предсказанию победителей боёв UFC на основе машинного обучения.

## 📋 Описание

Этот проект направлен на создание модели машинного обучения для прогнозирования исходов боёв Ultimate Fighting Championship (UFC). Проект использует исторические данные о боях, статистику бойцов и коэффициенты букмекеров для обучения модели.

## 📁 Структура проекта

```
├── UFC_Pet_Project.ipynb    # Jupyter Notebook с анализом данных и обучением модели
├── ufc-master.csv           # Датасет с историческими данными о боях UFC
└── README.md                # Документация проекта
```

## 📊 Данные

Датасет `ufc-master.csv` содержит подробную информацию о боях UFC, включая:

- **Информация о бойцах**: имя, стойка, рост, размах рук, вес, возраст
- **Статистика бойцов**: 
  - Среднее количество значимых ударов в минуту
  - Процент точности ударов
  - Среднее количество попыток сабмишенов
  - Среднее количество тейкдаунов и их процент
  - Текущие серии побед/поражений
  - Общая статистика побед (KO/TKO, сабмишен, решение)
- **Информация о бое**: дата, место проведения, весовая категория, титульный бой
- **Коэффициенты**: букмекерские коэффициенты и ожидаемая ценность (EV)
- **Результат**: победитель, способ завершения, раунд и время завершения

## 🚀 Использование

1. Откройте Jupyter Notebook:
   ```bash
   jupyter notebook UFC_Pet_Project.ipynb
   ```

2. Запустите ячейки последовательно для:
   - Загрузки и предварительной обработки данных
   - Разведочного анализа данных (EDA)
   - Подготовки признаков
   - Обучения модели
   - Оценки результатов

## 🛠 Технологии

- Python 3
- Pandas (работа с данными)
- NumPy (численные вычисления)
- Scikit-learn (машинное обучение)
- Matplotlib/Seaborn (визуализация)
- Jupyter Notebook

## 📈 Возможности модели

Модель может предсказывать:
- Победителя боя (Red Corner или Blue Corner)
- Вероятность победы каждого бойца
